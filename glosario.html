Glosario


HTML:
HTML se refiere a Hyper Text Markup Language. Este es el código utilizado para agregarle texto, imágenes y otro contenido a una página web.
WWW:
La World Wide Web es un sistema de información masiva en la cual documentos y recursos son compartidos entre un numero indefinible de computadoras.
Web Page:
Un Web Page es un documento escrito en HTML que puedes ver utilizando un web browser.
Internet:
La Internet es una red cuya función es conectar computadoras alrededor del mundo para que estas puedan compartir información y comunicarse.
UTF-8:
El Unicode Transformation Format 8 es un formato del sistema de codificación de Unicode que usa 1 a 4 bytes.
Web Browser
Un Web Browser es una herramienta de software que permite que un usuario pueda acceder a la world wide web y pueda leer documentos subidos a la web.
TCP/IP
El modelo TCP/IP es una serie de protocolos de comunicación que son seguidos por computadoras que se comunican a larga distancia.
Elemento
Un elemento de HTML se define como caracteres encerrados entre 2 brackets. Estos caracteres se llaman "tags".
ARPANET
El Advanced Research Projects Agency Network, es una red de computadoras conectadas por líneas telefónicas a través de diferentes instituciones educacionales.
Computadora
Una computadora es una maquina o dispositivo que realiza procesos, cálculos y operaciones a través de instrucciones y estándares presentados para el software y hardware.
